# i-wanna-be-yours-figma
Ilustração tipográfica baseada na música “I Wanna Be Yours” (Arctic Monkeys), desenvolvida no Figma para a disciplina de Design.
Este repositório contém o meu primeiro trabalho de Ilustração Tipográfica, desenvolvido para a disciplina.

O objetivo era criar uma composição visual inspirada numa música.
A música que escolhi foi “I Wanna Be Yours” – Arctic Monkeys.

O projeto foi desenvolvido no Figma, de forma simples, seguindo as instruções do professor.

Este também é o meu primeiro trabalho deste tipo, então ainda estou a aprender e a explorar novas formas de criar.
